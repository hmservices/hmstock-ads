Connect Your API Key

// one-time setup

To use HMSTOCK Ad Lab you need an Anthropic API key. Takes ~5 minutes.

Steps:

  1. Go to console.anthropic.com
  2. Sign in (or create account)
  3. Click API Keys in left sidebar → Create Key
  4. Add credit ($5-$10 covers months of use)
  5. Copy the key (starts with sk-ant-...) and paste below
Cost: ~$0.05-$0.15 per ad batch. $10 = roughly 100 generations.

Your key stays in your browser only (localStorage). Never sent anywhere except Anthropic's API.
